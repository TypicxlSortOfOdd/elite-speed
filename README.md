## Elite Speed
_A mod that adds high-speed rail options_

### Support
1.1.4-rc (beta)  AKA  1.1.5-rc (beta)

_This mod was built to support versions up to and around this version_

### Shinkansen
Also known as the Japanese Bullet Train, Shinkansen is one of the most recognizable passenger trains in existence.
- Runs up to 196.8 Mi/H ( 316.9 Km/H | 88 m/s )
- Very high operating costs
- 6 to 16 cars; 2 car set ( 6 | 8   10   12   14 | 16 )

### About
This mod is in testing.
- It's loosely based off of modified real-world figures
- These trains were designed to be used sparingly
- My first proper mod for Subway Builder
